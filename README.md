## Installation
To install **OnlineJobPortal**, follow these steps:

Linux and macOS:

```bash
sudo git clone https://github.com/UmbrellaSkiies/OnlineJobPortal.git
```

Windows:

```bash
git clone https://github.com/UmbrellaSkiies/OnlineJobPortal.git
```
